Error codes and what they mean

  • [ERR01]: 'prep' folder not found at the specified location.
  • [ERR02]: EFI System Partition not found.
  • [ERR06]: Failed to create 'targz' directory.
  • [ERR07]: Failed to create 'chrome_data' directory.
  • [ERR08]: Failed to execute xcopy command.
  • [ERR09]: Failed to copy Chrome data.
  • [ERR10]: Failed to write username to file.
